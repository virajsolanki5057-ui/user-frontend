# User List Creator

A modern React application built with **Vite** and **Tailwind CSS** that displays a dynamic list of users integrated from a REST API.

## 🛠️ Tech Stack
- **React 18** (UI Library)
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **JSONPlaceholder** (API Source)
